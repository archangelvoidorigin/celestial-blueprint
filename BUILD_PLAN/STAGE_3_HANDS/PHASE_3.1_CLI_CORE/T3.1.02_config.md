# TASK T3.1.02 — `cli/config.py` — env + TOML config loader

- **Stage / Phase**: Stage 3 — Hands / Phase 3.1
- **Depends on**: T3.1.01
- **Produces**: `backend/cli/config.py`

## Context

The `cb` CLI needs to know the API URL and auth token. Config is resolved in priority order: environment variables override a `~/.config/celestial/config.toml` file, which overrides compiled-in defaults. `tomllib` is Python 3.11+ stdlib (no extra dep). The config file uses a `[default]` section with keys `api_url` and `token`. If the file does not exist, config falls back to defaults silently.

## Objective

Create `backend/cli/config.py` exposing a `get_config() -> dict` function that returns `{"api_url": str, "token": str}` with the correct priority merge.

## Instructions

1. Create `backend/cli/config.py` with exactly:
   ```python
   """CLI configuration loader.

   Priority (highest to lowest):
     1. Environment variables: CELESTIAL_API_URL, CELESTIAL_AUTH_TOKEN
     2. ~/.config/celestial/config.toml  [default] section
     3. Compiled-in defaults
   """
   from __future__ import annotations

   import os
   import tomllib
   from pathlib import Path

   _DEFAULTS = {
       "api_url": "http://localhost:8787",
       "token": "",
   }

   _CONFIG_PATH = Path.home() / ".config" / "celestial" / "config.toml"


   def _load_toml() -> dict:
       """Return the [default] section from the config file, or {} if absent."""
       if not _CONFIG_PATH.exists():
           return {}
       try:
           with _CONFIG_PATH.open("rb") as fh:
               data = tomllib.load(fh)
           return data.get("default", {})
       except Exception:
           return {}


   def get_config() -> dict:
       """Return resolved config dict with keys: api_url, token."""
       file_cfg = _load_toml()
       return {
           "api_url": os.environ.get(
               "CELESTIAL_API_URL",
               file_cfg.get("api_url", _DEFAULTS["api_url"]),
           ),
           "token": os.environ.get(
               "CELESTIAL_AUTH_TOKEN",
               file_cfg.get("token", _DEFAULTS["token"]),
           ),
       }
   ```
