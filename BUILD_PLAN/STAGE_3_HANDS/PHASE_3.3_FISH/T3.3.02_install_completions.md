# TASK T3.3.02 — `cli/install_completions.sh` + dev.fish reminder

- **Stage / Phase**: Stage 3 — Hands / Phase 3.3
- **Depends on**: T3.3.01
- **Produces**: `backend/cli/install_completions.sh`, edit to `Celestial_Blueprint/dev.fish`

## Context

The `cb.fish` completions file (T3.3.01) lives in `backend/cli/completions/`. For fish to pick them up at every shell start, a symlink must exist at `~/.config/fish/completions/cb.fish`. The install script creates that symlink. The `dev.fish` startup script (created in T1.1.08) should print a one-line reminder if the symlink does not already exist, so developers know to run the script the first time they set up the project.

The `dev.fish` file lives at `Celestial_Blueprint/dev.fish` (project root, not in `backend/`).

## Objective

Create `backend/cli/install_completions.sh` and add a symlink-check reminder block to `dev.fish`.

## Instructions

1. Create `backend/cli/install_completions.sh` with exactly:
   ```bash
   #!/usr/bin/env bash
   set -euo pipefail

   SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
   COMPLETIONS_SRC="$SCRIPT_DIR/completions/cb.fish"
   COMPLETIONS_DIR="$HOME/.config/fish/completions"
   COMPLETIONS_DST="$COMPLETIONS_DIR/cb.fish"

   if [ ! -f "$COMPLETIONS_SRC" ]; then
       echo "Error: $COMPLETIONS_SRC not found" >&2
       exit 1
   fi

   mkdir -p "$COMPLETIONS_DIR"

   if [ -L "$COMPLETIONS_DST" ]; then
       echo "Symlink already exists: $COMPLETIONS_DST"
   elif [ -f "$COMPLETIONS_DST" ]; then
       echo "Warning: $COMPLETIONS_DST exists as a regular file. Remove it manually and re-run." >&2
       exit 1
   else
       ln -s "$COMPLETIONS_SRC" "$COMPLETIONS_DST"
       echo "Installed: $COMPLETIONS_DST -> $COMPLETIONS_SRC"
   fi

   echo "cb fish completions are ready. Start a new fish shell or run: source $COMPLETIONS_SRC"
   ```

2. Make the script executable:
   ```bash
   chmod +x /home/void-yggdrasil/CHAOS_DAO_FORGE/00_CORTEX/01_CREATION_REALM/Celestial_Blueprint/backend/cli/install_completions.sh
   ```

3. Edit `Celestial_Blueprint/dev.fish`. Find the end of the file and append the following block. If the file does not exist yet (it is created in T1.1.08), create it with only this block for now — T1.1.08 will complete the rest:
   ```fish
   # --- cb completions reminder ---
   set _cb_comp ~/.config/fish/completions/cb.fish
   if not test -L $_cb_comp
       echo "[ cb completions not installed ] Run: bash backend/cli/install_completions.sh"
   end
   ```

   To append to the existing file, add the block at the very end, separated by a blank line from whatever was already there.
