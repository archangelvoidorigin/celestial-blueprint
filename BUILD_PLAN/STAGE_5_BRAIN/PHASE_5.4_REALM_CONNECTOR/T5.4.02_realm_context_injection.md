# TASK T5.4.02 — Inject `realm_context` into `GET /api/blueprints/{id}`

- **Stage / Phase**: Stage 5 — Brain / Phase 5.4
- **Depends on**: T5.4.01
- **Produces**: edit to `backend/app/api/blueprints.py`, edit to `docs/API.md`

## Context

`get_realm_context` exists (T5.4.01) but nothing calls it yet. The blueprint GET endpoint (`app/api/blueprints.py`) currently returns the raw blueprint dict. This task edits `get_blueprint()` to append a `realm_context` key: if the blueprint's frontmatter has a `realm` field, it calls `get_realm_context(realm)` and merges the result into the response. If `realm` is absent or `get_realm_context` returns `None`, the key is omitted (no `null` in the response). This is a non-breaking additive change.

## Objective

Edit `app/api/blueprints.py`'s `get_blueprint` endpoint to inject `realm_context` from the realm service.

## Instructions

1. Edit `backend/app/api/blueprints.py`:

   a. Add import at the top (after existing imports):
      ```python
      from app.services.realm import get_realm_context
      ```

   b. Replace the `get_blueprint` function body:
      ```python
      @router.get("/{bid}")
      def get_blueprint(bid: str):
          bp = store.get_blueprint(bid)
          if bp is None:
              raise HTTPException(status_code=404, detail="blueprint not found")
          realm = bp.get("frontmatter", {}).get("realm", "")
          if realm:
              ctx = get_realm_context(realm)
              if ctx is not None:
                  bp["realm_context"] = ctx
          return bp
      ```

2. Edit `docs/API.md`: under `## Stage 5 — Brain`, append:
   ```markdown
   ### `GET /api/blueprints/{id}` — `realm_context` field (Stage 5 addition)
   If the blueprint's `realm` frontmatter field resolves to a readable `smart-folder.md`, the response includes:
   ```json
   "realm_context": {
     "realm_path": "/abs/path",
     "title": "Realm Title",
     "phase": "current phase",
     "status": "active",
     "agents": [],
     "tags": [],
     "last_modified": "2026-06-25T10:00:00"
   }
   ```
   Field is absent if `realm` is unset or path does not resolve.
   ```
