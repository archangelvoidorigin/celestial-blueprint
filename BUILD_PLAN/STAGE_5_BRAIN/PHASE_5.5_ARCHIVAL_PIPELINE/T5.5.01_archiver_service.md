## Context

Hard rule #4: never delete blueprints — archiving flips status and moves the file. The existing `POST /api/blueprints/{id}/archive` endpoint (T1.4.02) sets DB status to `archived` and rewrites the markdown file in `blueprints/` but does NOT move it. Stage 5 adds `archiver.py` which moves the file from `blueprints/` to `CELESTIAL_ARCHIVE_DIR` (defaults to `02_OMNI` extraction realm). If `CELESTIAL_ARCHIVE_DIR` is not set, the file stays in place (the existing behaviour is preserved — no breakage).

## Objective

Create `app/services/archiver.py` with `archive_blueprint_file(bid)` that moves the blueprint markdown to the archive directory.

## Instructions

1. Edit `backend/app/core/config.py`: add to the `Settings` dataclass (frozen `@dataclass`, NOT Pydantic — use `field(default_factory=...)`, matching the existing fields; `field` is already imported):
   ```python
   archive_dir: str = field(
       default_factory=lambda: os.environ.get("CELESTIAL_ARCHIVE_DIR", "")
   )
   ```

2. Edit `backend/.env.example`: append:
   ```
   CELESTIAL_ARCHIVE_DIR=
   ```
   (Leave empty — user fills in e.g. `/home/void-yggdrasil/CHAOS_DAO_FORGE/02_OMNI/archived-blueprints`)

3. Create `backend/app/services/archiver.py` with exactly:
   ```python
   """Archival pipeline: move blueprint markdown to the OMNI archive dir."""
   from __future__ import annotations

   import shutil
   from pathlib import Path

   from app.core.config import settings
   from app.services.files import blueprint_path


   def archive_blueprint_file(bid: str) -> str | None:
       """Move {blueprints_dir}/{bid}.md to {archive_dir}/{bid}.md.

       Returns the destination path string, or None if archive_dir is not
       configured (file stays in place — no error).
       """
       if not settings.archive_dir:
           return None

       src = blueprint_path(bid)
       if not src.exists():
           return None

       dest_dir = Path(settings.archive_dir)
       dest_dir.mkdir(parents=True, exist_ok=True)
       dest = dest_dir / src.name
       shutil.move(str(src), str(dest))
       return str(dest)
   ```
