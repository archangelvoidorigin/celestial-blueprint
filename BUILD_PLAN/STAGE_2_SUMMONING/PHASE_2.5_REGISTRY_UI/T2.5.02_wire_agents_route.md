# TASK T2.5.02 — Wire Agents route into App.tsx + a link from the list page

- **Stage / Phase**: Stage 2 — Summoning / Phase 2.5
- **Depends on**: T2.5.01
- **Produces**: `frontend/src/App.tsx` (Agents import + route added), `frontend/src/pages/BlueprintList.tsx` (Agents link added to header)

## Context

`frontend/src/App.tsx` (Stage 1, T1.6.08) is a **bare `<Routes>` tree with no nav bar/header**.
It contains exactly three routes:
```tsx
import { Routes, Route, Navigate } from "react-router-dom";
import BlueprintList from "./pages/BlueprintList";
import Canvas from "./pages/Canvas";

export default function App() {
  return (
    <Routes>
      <Route path="/" element={<BlueprintList />} />
      <Route path="/blueprint/:id" element={<Canvas />} />
      <Route path="*" element={<Navigate to="/" replace />} />
    </Routes>
  );
}
```
There is no `<nav>` to add a link to. So this task (1) adds the `/agents` route — placed
**before** the `*` catch-all so it isn't swallowed by the redirect — and (2) adds a small
"Agents" link into the header of `BlueprintList.tsx` (the home page) so the registry is
reachable. Use Tailwind theme-token classes (`text-muted`, `hover:text-ink`), matching the
rest of the app.

## Objective

Add the `/agents` route to `App.tsx` and an "Agents" link to the `BlueprintList` header.

## Instructions

### 1. Edit `frontend/src/App.tsx`

a. Add the page import alongside the existing page imports:
```tsx
import Agents from "./pages/Agents";
```

b. Add the route **immediately before** the `*` catch-all route:
```tsx
      <Route path="/agents" element={<Agents />} />
```
The resulting `<Routes>` block must be:
```tsx
    <Routes>
      <Route path="/" element={<BlueprintList />} />
      <Route path="/blueprint/:id" element={<Canvas />} />
      <Route path="/agents" element={<Agents />} />
      <Route path="*" element={<Navigate to="/" replace />} />
    </Routes>
```

### 2. Edit `frontend/src/pages/BlueprintList.tsx`

a. Add `Link` to the existing react-router-dom import:
```tsx
import { useNavigate, Link } from "react-router-dom";
```

b. In the header row, wrap the existing "+ New" button so an "Agents" link sits beside it.
   Replace this block:
```tsx
         <div className="mb-6 flex items-center justify-between">

             className="rounded bg-accent px-4 py-1.5 text-sm font-medium text-white hover:bg-violet-500"
```

