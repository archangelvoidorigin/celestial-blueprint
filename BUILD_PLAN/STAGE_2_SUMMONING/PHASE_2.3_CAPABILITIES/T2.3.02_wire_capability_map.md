# TASK T2.3.02 — Wire capability map into scanner detectors

- **Stage / Phase**: Stage 2 — Summoning / Phase 2.3
- **Depends on**: T2.3.01
- **Produces**: `backend/agents/scanner.py` — each detector uses `get_capabilities()` as baseline; unknown custom agents run through `validate_capabilities()`

## Context

Currently each detector in `scanner.py` hardcodes its own capabilities list. That's fine for
known agents but diverges from `capability_map.py`. This task makes each detector call
`get_capabilities(agent_id)` as its default, so `capability_map.py` is the single source of
truth. Custom agents from `detect_custom_agents()` get their declared capabilities validated
against `CAPABILITY_VOCAB` to strip invalid strings.

## Objective

Update `backend/agents/scanner.py` to import and use `get_capabilities` and
`validate_capabilities` from `agents.capability_map` in each detector.

## Instructions

1. Open `backend/agents/scanner.py`.
2. Add at the top (after existing imports):
```python
from agents.capability_map import get_capabilities, validate_capabilities
```

3. In each single-agent detector (`detect_claude_code`, `detect_opencode`, `detect_codex`,
   `detect_antigravity`, `detect_hermes`), replace the hardcoded `capabilities=[...]` line
   in the `agent_entry(...)` call with:
   ```python
   capabilities=get_capabilities("<agent-id>"),
   ```
   For example in `detect_claude_code`:
   ```python
   return agent_entry(
       id="claude-code",
       name="Claude Code",
       capabilities=get_capabilities("claude-code"),   # ← changed
       activation_cmd=binary,
       status="active",
       version=version,
   )
   ```
   Do the same replacement for `detect_opencode` → `"opencode"`, `detect_codex` → `"codex"`,
   `detect_antigravity` → `"antigravity"`, `detect_hermes` → `"hermes"`.
   For `detect_ollama`, keep `get_capabilities("ollama")` as the base.

4. In `detect_custom_agents`, after loading `data.get("capabilities", [])`, wrap it:
   ```python
   raw_caps = data.get("capabilities", [])
   capabilities=validate_capabilities(raw_caps),
   ```

## Acceptance Criteria

- [ ] `scanner.py` imports `get_capabilities` without error
- [ ] `detect_claude_code()["capabilities"]` matches `DEFAULT_CAPABILITIES["claude-code"]`
- [ ] Changing a value in `capability_map.py` is reflected in scanner output without editing `scanner.py`
