## Context

The routing service (T2.4.01) is a pure function in `app/services/router.py`. This task
wraps it in a FastAPI router with one endpoint: `POST /api/route`. The request body carries
`task_type` and optional `context`. The response is the routing decision dict. This file is
named `app/api/router.py` — do not confuse it with FastAPI's `APIRouter` class (which it
uses internally).

## Objective

Create `backend/app/api/router.py` with a single `POST /api/route` endpoint.

## Instructions

Create `backend/app/api/router.py` with this content:

```python
"""
Celestial Blueprint — Route API
POST /api/route — resolve best agent for a task type.
Auth is applied globally at include time (main.py), matching the other routers.
"""
from __future__ import annotations

from fastapi import APIRouter
from pydantic import BaseModel

from app.services.router import route_task

router = APIRouter(prefix="/api", tags=["routing"])


class RouteRequest(BaseModel):
    task_type: str
    context: str = ""


class RouteResponse(BaseModel):
    agent_id: str
    agent_name: str
    activation_cmd: str
    task_type: str
    reason: str


@router.post("/route", response_model=RouteResponse)
def resolve_route(req: RouteRequest) -> RouteResponse:
    """
    Given a task_type (e.g. 'code_gen', 'research'), return the best
    available agent and its activation command.
    """
    decision = route_task(req.task_type, req.context)
    return RouteResponse(**decision)
```
