## Context

OpenCode is a TUI-based AI coding assistant. It can be installed as the `opencode` binary
(npm global or direct). Its config lives at `~/.config/opencode/config.json` and skills at
`~/.config/opencode/skills/` (or `~/.opencode/skills/`). On this system an `opus` alias wraps
the opencode-memory version, but the core binary is `opencode`. Append one detector function
to `backend/agents/scanner.py`.

## Objective

Append `detect_opencode() -> dict | None` to `backend/agents/scanner.py`.

## Instructions

1. Open `backend/agents/scanner.py`.
2. Append after `detect_claude_code`:

```python
def detect_opencode() -> Optional[dict]:
    """Detect OpenCode TUI agent."""
    binary = detect_binary("opencode")
    if not binary:
        return None

    version = ""
    try:
        import subprocess
        result = subprocess.run(
            ["opencode", "--version"], capture_output=True, text=True, timeout=5
        )
        version = result.stdout.strip().split("\n")[0]
    except Exception:
        pass

    return agent_entry(
        id="opencode",
        name="OpenCode",
        capabilities=["code_gen", "file_ops", "plan", "mcp_tools"],
        activation_cmd=binary,
        status="active",
        version=version,
    )
```
