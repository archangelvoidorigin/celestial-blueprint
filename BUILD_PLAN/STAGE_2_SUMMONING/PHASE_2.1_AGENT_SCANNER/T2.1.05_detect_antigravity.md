## Context

Google's Antigravity (also called Gemini CLI) is an agentic coding assistant installed as the
`antigravity` or `gemini` binary via npm global or direct download. Its config lives at
`~/.config/antigravity/` or `~/.antigravity/`. The OMNI docs note 1,460+ skills and MCP stack
support. Append one detector to `backend/agents/scanner.py`.

## Objective

Append `detect_antigravity() -> dict | None` to `backend/agents/scanner.py`. Checks for both
`antigravity` and `gemini` binary names.

## Instructions

1. Open `backend/agents/scanner.py`.
2. Append after `detect_codex`:

```python
def detect_antigravity() -> Optional[dict]:
    """Detect Google Antigravity / Gemini CLI."""
    binary = detect_binary("antigravity") or detect_binary("gemini")
    if not binary:
        return None

    version = ""
    try:
        import subprocess
        result = subprocess.run(
            [binary, "--version"], capture_output=True, text=True, timeout=5
        )
        version = result.stdout.strip().split("\n")[0]
    except Exception:
        pass

    return agent_entry(
        id="antigravity",
        name="Google Antigravity / Gemini CLI",
        capabilities=["code_gen", "file_ops", "web_search", "plan", "mcp_tools"],
        activation_cmd=binary,
        status="active",
        version=version,
    )
```
