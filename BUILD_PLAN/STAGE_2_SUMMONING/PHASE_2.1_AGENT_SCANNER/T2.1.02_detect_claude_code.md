## Context

Claude Code is the Anthropic CLI tool for agentic coding. On this system it is installed as
the `claude` binary. It uses `~/.claude/` for config, `~/.claude/skills/` for skills, and
`~/.mcp-lazy/servers.json` for MCP server registration. This task appends one detector
function to `backend/agents/scanner.py` (created in T2.1.01).

## Objective

Append `detect_claude_code() -> dict | None` to `backend/agents/scanner.py`. It returns an
agent entry if the `claude` binary is found, else `None`.

## Instructions

1. Open `backend/agents/scanner.py`.
2. Append the following function after the `agent_entry` helper section comment:

```python
def detect_claude_code() -> Optional[dict]:
    """Detect Claude Code (Anthropic CLI)."""
    binary = detect_binary("claude")
    if not binary:
        return None

    # Try to get version
    version = ""
    try:
        import subprocess
        result = subprocess.run(
            ["claude", "--version"], capture_output=True, text=True, timeout=5
        )
        version = result.stdout.strip().split("\n")[0]
    except Exception:
        pass

    return agent_entry(
        id="claude-code",
        name="Claude Code",
        capabilities=["code_gen", "file_ops", "mcp_tools", "plan", "critique"],
        activation_cmd=binary,
        status="active",
        version=version,
    )
```
