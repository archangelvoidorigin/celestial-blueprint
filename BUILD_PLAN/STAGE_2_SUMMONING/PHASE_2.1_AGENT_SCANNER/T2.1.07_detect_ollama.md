## Context

Ollama is a local model server that exposes a REST API at `http://localhost:11434`. It runs
as a background process/service with the `ollama` binary. To detect it properly: check the
binary exists AND probe the REST endpoint `GET /api/tags` to list available models. The
system memory notes mention it was previously OOM-killed (7.5GB RSS) — it may not always be
running. Append one detector to `backend/agents/scanner.py`.

## Objective

Append `detect_ollama() -> dict | None` to `backend/agents/scanner.py`. Checks binary AND
probes the REST API for available models. Status is `"active"` only if the API responds.

## Instructions

1. Open `backend/agents/scanner.py`.
2. Append after `detect_hermes`:

```python
def detect_ollama() -> Optional[dict]:
    """Detect Ollama local model server."""
    binary = detect_binary("ollama")
    if not binary:
        return None

    # Probe REST endpoint for model list
    models: list[str] = []
    status = "inactive"
    try:
        import urllib.request
        with urllib.request.urlopen(
            "http://localhost:11434/api/tags", timeout=2
        ) as resp:
            data = json.loads(resp.read())
            models = [m["name"] for m in data.get("models", [])]
            status = "active"
    except Exception:
        pass  # offline or not started — still register it

    version = ""
    try:
        import subprocess
        result = subprocess.run(
            ["ollama", "--version"], capture_output=True, text=True, timeout=5
        )
        version = result.stdout.strip().split("\n")[0]
    except Exception:
        pass

    capabilities = ["code_gen", "summarise", "research", "critique"]
    entry = agent_entry(
        id="ollama",
        name="Ollama",
        capabilities=capabilities,
        activation_cmd=binary,
        status=status,
        version=version,
    )
    entry["models"] = models  # extend with available models list
    return entry
```
