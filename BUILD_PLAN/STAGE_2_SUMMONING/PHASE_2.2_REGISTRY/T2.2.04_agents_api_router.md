## Context

All API routers live in `backend/app/api/`. Each router is an `APIRouter` instance with a
full `/api/...` prefix that gets included into `app/main.py`. This task creates the agents
router with six endpoints. The rescan endpoint (`POST /api/agents/scan`) calls
`agents.scanner.scan_all()` and upserts all returned entries into the DB.

**Auth note**: Stage 1 (T1.7.02) applies `require_auth` *globally at include time* — i.e.
`app.include_router(router, dependencies=[Depends(require_auth)])` in `main.py` — and the
route handlers themselves carry no auth dependency. Match that pattern: this router declares
NO per-route auth. The auth gate is wired in T2.2.05 when the router is included.

## Objective

Create `backend/app/api/agents.py` with six endpoints: list, get, create, patch, delete,
and rescan.

## Instructions

Create `backend/app/api/agents.py` with this content:

```python
"""
Celestial Blueprint — Agents API Router
Endpoints under /api/agents. Auth is applied globally at include time (main.py).
"""
from __future__ import annotations

from fastapi import APIRouter, HTTPException, Query

from app.models.schemas import Agent, AgentCreate, AgentPatch
from app.services.agent_store import (
    delete_agent,
    get_agent,
    list_agents,
    patch_agent,
    upsert_agent,
)

router = APIRouter(prefix="/api/agents", tags=["agents"])


@router.get("", response_model=list[Agent])
def get_agents(status: str | None = Query(None)):
    """List all registered agents, optionally filtered by status."""
    return list_agents(status=status)


@router.post("/scan", response_model=list[Agent])
def trigger_scan():
    """
    Run the agent scanner, upsert all discovered agents into the DB,
    and return the updated registry. Declared before /{agent_id} so the
    literal /scan path is never shadowed by the dynamic route.
    """
    from agents.scanner import scan_all

    discovered = scan_all()
    for entry in discovered:
        upsert_agent(
            AgentCreate(
                id=entry["id"],
                name=entry["name"],
                capabilities=entry.get("capabilities", []),
                activation_cmd=entry.get("activation_cmd", ""),
                status=entry.get("status", "unknown"),
                version=entry.get("version", ""),
                models=entry.get("models", []),
            )
        )
    return list_agents()


@router.get("/{agent_id}", response_model=Agent)
def get_agent_by_id(agent_id: str):
    agent = get_agent(agent_id)
    if not agent:
        raise HTTPException(status_code=404, detail="Agent not found")
    return agent


@router.post("", response_model=Agent, status_code=201)
def register_agent(data: AgentCreate):
    """Manually register or re-register an agent."""
    return upsert_agent(data)


@router.patch("/{agent_id}", response_model=Agent)
def update_agent(agent_id: str, patch: AgentPatch):
    updated = patch_agent(agent_id, patch)
    if not updated:
        raise HTTPException(status_code=404, detail="Agent not found")
    return updated


@router.delete("/{agent_id}", status_code=204)
def remove_agent(agent_id: str):
    if not delete_agent(agent_id):
        raise HTTPException(status_code=404, detail="Agent not found")
```
