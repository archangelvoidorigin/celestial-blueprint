# TASK T2.2.01 — Agents table schema + init_agents()

- **Stage / Phase**: Stage 2 — Summoning / Phase 2.2
- **Depends on**: T2.1.08
- **Produces**: `backend/app/models/agents_schema.sql` (new), updated `backend/app/core/db.py` (`init_agents()` appended)

## Context

Celestial Blueprint's SQLite database is initialized by `backend/app/core/db.py`. Stage 1
defined three functions there — `get_conn()`, `init_db()`, and a `connection()` context
manager — and `init_db()` runs `app/models/schema.sql`, which contains exactly 8 core tables
(a count Stage 1 asserts in its tests). Stage 2 needs an `agents` table for the registry.

**Do NOT modify `schema.sql` and do NOT rewrite `init_db()`** — that would break Stage 1's
8-table assertion and every module that imports `get_conn`. Instead, add a separate
`agents_schema.sql` and an additive `init_agents()` function. The DB path comes from
`settings.db_path` (overridable in tests via the `CELESTIAL_DB_PATH` env var).

## Objective

Create `backend/app/models/agents_schema.sql` and append an `init_agents()` function to
`backend/app/core/db.py` that applies it idempotently.

## Instructions

1. Create `backend/app/models/agents_schema.sql` with exactly:

```sql
-- Celestial Blueprint — agents registry (Stage 2).
-- Separate from the core 8-table schema; applied by init_agents().
PRAGMA foreign_keys = ON;

CREATE TABLE IF NOT EXISTS agents (
    id             TEXT PRIMARY KEY,
    name           TEXT NOT NULL,
    capabilities   TEXT NOT NULL DEFAULT '[]',        -- JSON array of strings
    activation_cmd TEXT,
    status         TEXT NOT NULL DEFAULT 'unknown',   -- active/inactive/unknown
    version        TEXT NOT NULL DEFAULT '',
    models         TEXT NOT NULL DEFAULT '[]',        -- JSON array (Ollama models etc.)
    discovered_at  TEXT NOT NULL DEFAULT (datetime('now')),
    updated_at     TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX IF NOT EXISTS idx_agents_status ON agents(status);
```

2. Open `backend/app/core/db.py`. It already has `SCHEMA_PATH`, `get_conn()`, `init_db()`,
   and `connection()`. Append the following at the end of the file (do not touch the existing
   functions):

```python
AGENTS_SCHEMA_PATH = Path(__file__).resolve().parents[1] / "models" / "agents_schema.sql"


def init_agents() -> None:
    """Create the agents table if it does not exist (idempotent). Additive to init_db()."""
    schema = AGENTS_SCHEMA_PATH.read_text()
    conn = get_conn()
    try:
        conn.executescript(schema)
        conn.commit()
    finally:
        conn.close()
```
