   # Celestial Blueprint — Keybinds Reference

   All keybinds require Hyprland with `hypr/celestial.conf` sourced (run `hypr/link_hypr.sh` once after install — see T6.1.01).

   ## Quick-launch

   | Keybind | Action |
   |---------|--------|
   | `Super + P` | Open a floating terminal (shows `cb blueprint list`, then a shell) |
   | `Super + Shift + P` | Open the Celestial Blueprint frontend at `http://localhost:5173` |

   ## Workspace toggle

   | Keybind | Action |
   |---------|--------|
   | `Super + C` | Toggle the Celestial special scratchpad workspace |

   (`Super + C` is defined in `hypr/celestial.conf` — added by T6.2.01.)

   ## Setup

   ```bash
   # One-time: source the config fragment from hyprland.conf
   bash hypr/link_hypr.sh

   # Reload Hyprland to activate
   hyprctl reload
   ```

   ## Customise

   Edit `hypr/celestial.conf` to change modifiers or terminal emulator.
   - Replace `alacritty` with your emulator if it supports `--class` and `-e`.
   - Replace `$mainMod` with a different modifier key if needed.
   - Run `hyprctl reload` after any edit (no restart needed).
   ```
