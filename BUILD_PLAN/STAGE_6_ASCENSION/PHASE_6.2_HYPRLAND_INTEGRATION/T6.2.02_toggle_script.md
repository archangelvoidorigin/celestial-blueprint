# TASK T6.2.02 — `tools/toggle_celestial.sh`

- **Stage / Phase**: Stage 6 — Ascension / Phase 6.2
- **Depends on**: T1.8.01
- **Produces**: `tools/toggle_celestial.sh`

## Context

The keybind `$mainMod + C` calls `hyprctl dispatch togglespecialworkspace celestial` — but that only works if a window is already assigned to it. `toggle_celestial.sh` is the smarter version: if `alacritty --class celestial-plan` is NOT already running, it launches it; if it is running, it just toggles the scratchpad workspace. This means the keybind in T6.2.01 could also be changed to call this script via `exec` instead of `togglespecialworkspace` directly — the script handles both cases. Both approaches are valid; the script exists for users who prefer the smart launch approach.

## Objective

Create `tools/toggle_celestial.sh` that launches the Celestial terminal if not running, or toggles the scratchpad workspace.

## Instructions

1. Create `tools/toggle_celestial.sh` with exactly:
   ```bash
   #!/usr/bin/env bash
   # Toggle the Celestial Blueprint scratchpad terminal.
   # Launches alacritty with class=celestial-plan if not running;
   # otherwise toggles the special:celestial workspace.
   #
   # Usage: bind = $mainMod, C, exec, /path/to/toggle_celestial.sh
   # (Alternative to the direct togglespecialworkspace bind in celestial.conf.)

   CB_CLASS="celestial-plan"
   CB_WORKSPACE="celestial"
   CB_TITLE="Celestial Plan"

   # Check if a window with this class is already open
   if hyprctl clients -j | python3 -c "
   import json, sys
   clients = json.load(sys.stdin)
   running = any(c.get('class') == '$CB_CLASS' for c in clients)
   sys.exit(0 if running else 1)
   " 2>/dev/null; then
       # Window exists — toggle the scratchpad
       hyprctl dispatch togglespecialworkspace "$CB_WORKSPACE"
   else
       # Not running — launch it (windowrulev2 will float + assign it)
       alacritty \
           --class "$CB_CLASS" \
           --title "$CB_TITLE" \
           -e fish -c "cb blueprint list; echo; echo 'Celestial Blueprint — try: cb blueprint create \"My Plan\"'; exec fish" \
           &
   fi
   ```

2. Make it executable:
   ```bash
   chmod +x /home/void-yggdrasil/CHAOS_DAO_FORGE/00_CORTEX/01_CREATION_REALM/Celestial_Blueprint/tools/toggle_celestial.sh
   ```
