# TASK T6.3.01 — `tools/notify.sh` — desktop notification after CYCLE chronicle

- **Stage / Phase**: Stage 6 — Ascension / Phase 6.3
- **Depends on**: T1.8.01 (backend + chronicles endpoint)
- **Produces**: `tools/notify.sh`

## Context

After a planning session ends, the SKILL.md protocol saves a CYCLE chronicle. `notify.sh` queries the API for the most recent CYCLE chronicle of a given blueprint, extracts the `Next session:` line from its body, and fires a `dunstify` (or `notify-send`) desktop notification. It is designed to be called by Hermes, by the CLI `cb` after a CYCLE, or manually. Uses `httpx`-free curl+python3 — no venv required.

The CYCLE chronicle body follows this convention (from the methodology SKILL):
```
Next session: <short description of what to do next>
```
If that line is absent, the notification still fires with a generic message.

## Objective

Create `tools/notify.sh` that queries the last CYCLE chronicle for a blueprint and fires a desktop notification.

## Instructions

1. Create `tools/notify.sh` with exactly:
   ```bash
   #!/usr/bin/env bash
   # Fire a desktop notification summarising the last CYCLE chronicle.
   #
   # Usage:
   #   tools/notify.sh <blueprint-id>
   #   CELESTIAL_API_URL=http://localhost:8787 tools/notify.sh bp_2026-06-25_my-plan
   #
   # Dependencies: curl, python3, dunstify (or notify-send as fallback)

   set -euo pipefail

   API_URL="${CELESTIAL_API_URL:-http://localhost:8787}"
   BID="${1:-}"

   if [[ -z "$BID" ]]; then
       echo "Usage: $0 <blueprint-id>" >&2
       exit 1
   fi

   # Fetch chronicles for the blueprint
   CHRONICLES=$(curl -sf "$API_URL/api/blueprints/$BID/chronicles" 2>/dev/null || echo "[]")

   # Extract last CYCLE chronicle body (JSON piped in on stdin)
   RESULT=$(printf '%s' "$CHRONICLES" | python3 -c '
   import json, sys

   try:
       chronicles = json.loads(sys.stdin.read())
   except Exception:
       chronicles = []

   cycles = [c for c in chronicles if c.get("kind") == "cycle"]
   if not cycles:
       print("NO_CYCLE")
       sys.exit(0)

   last = sorted(cycles, key=lambda c: c.get("id", 0))[-1]
   body = last.get("body", "")

   next_line = ""
   for line in body.splitlines():
       if line.strip().lower().startswith("next session:"):
           next_line = line.split(":", 1)[1].strip()
           break

   print(f"Next: {next_line}" if next_line else "Planning cycle complete.")
   ')

   if [[ "$RESULT" == "NO_CYCLE" ]]; then
       echo "No CYCLE chronicles found for $BID — no notification sent."
       exit 0
   fi

   # Blueprint name (best-effort)
   BP_NAME=$(curl -sf "$API_URL/api/blueprints/$BID" 2>/dev/null \
     | python3 -c "import json,sys; d=json.load(sys.stdin); print(d['frontmatter'].get('name', '$BID'))" 2>/dev/null \
     || echo "$BID")

   TITLE="Celestial Blueprint — $BP_NAME"
   BODY="$RESULT"

   # Fire notification (dunstify preferred; fallback to notify-send)
   if command -v dunstify &>/dev/null; then
       dunstify -u normal -t 8000 -i dialog-information "$TITLE" "$BODY"
   elif command -v notify-send &>/dev/null; then
       notify-send -t 8000 "$TITLE" "$BODY"
   else
       echo "No notification daemon found (dunstify/notify-send). Message: $TITLE — $BODY" >&2
   fi

   echo "Notified: $TITLE — $BODY"
   ```

