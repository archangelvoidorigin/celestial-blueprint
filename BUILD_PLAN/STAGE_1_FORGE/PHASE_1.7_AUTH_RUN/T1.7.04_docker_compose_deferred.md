## Context
A Docker Compose file is useful for running Celestial Blueprint on the phone-as-VPS or any container host, but it is NOT required for Stage 1 to be complete. This task is explicitly deferred: skip it unless you specifically need containerised deployment at this stage. A placeholder compose file is provided as a reference; mark the task done only after verifying both services start inside containers.

## Objective
Create a `docker-compose.yml` that runs the FastAPI backend and Vite-built frontend as two services.

## Instructions
> **Skip this task if you don't need Docker now.** The app runs fine with `fish dev.fish`. Return to this task if you want to deploy to a VPS or phone.

1. Create `docker-compose.yml` at the project root with exactly:
   ```yaml
   # Celestial Blueprint — container deployment (optional Stage 1 task)
   # Requirements: Docker, docker-compose v2
   version: "3.9"

   services:
     backend:
       build:
         context: ./backend
         dockerfile: Dockerfile
       env_file: ./backend/.env
       ports:
         - "8787:8787"
       volumes:
         - ./blueprints:/data/blueprints
         - ./chronicles:/data/chronicles
         - celestial-db:/data
       environment:
         CELESTIAL_DB_PATH: /data/celestial.db
         CELESTIAL_BLUEPRINTS_DIR: /data/blueprints
         CELESTIAL_CHRONICLES_DIR: /data/chronicles

     frontend:
       build:
         context: ./frontend
         dockerfile: Dockerfile
       ports:
         - "5173:80"
       depends_on:
         - backend

   volumes:
     celestial-db:
   ```
2. Create `backend/Dockerfile` with exactly:
   ```dockerfile
   FROM python:3.11-slim
   WORKDIR /app
   COPY requirements.txt .
   RUN pip install --no-cache-dir -r requirements.txt
   COPY . .
   CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8787"]
   ```
3. Create `frontend/Dockerfile` with exactly:
   ```dockerfile
   FROM node:20-alpine AS build
   WORKDIR /app
   COPY package*.json ./
   RUN npm ci
   COPY . .
   RUN npm run build

   FROM nginx:alpine
   COPY --from=build /app/dist /usr/share/nginx/html
   EXPOSE 80
   ```
