## Context
Celestial Blueprint is a local single-user tool, so auth is a single bearer token stored in `CELESTIAL_AUTH_TOKEN` (`.env`). When the token is set, every `/api/*` request (except `/api/health`) must present it as `Authorization: Bearer <token>`. When the env var is empty (dev without a token), the gate is open (useful for quick local testing). This module provides a FastAPI dependency `require_auth` that the routes use.

## Objective
Implement `app/core/auth.py` with a `require_auth` dependency that validates the bearer token from the environment.

## Instructions
1. Create `backend/app/core/auth.py` with exactly:
   ```python
   """Local single-user bearer token auth. Empty token = gate open (dev)."""
   from __future__ import annotations

   from fastapi import Depends, HTTPException, status
   from fastapi.security import HTTPAuthorizationCredentials, HTTPBearer

   from app.core.config import settings

   _bearer = HTTPBearer(auto_error=False)


   def require_auth(
       creds: HTTPAuthorizationCredentials | None = Depends(_bearer),
   ) -> None:
       """FastAPI dependency. Raises 401 when token is wrong; passes when empty."""
       expected = settings.auth_token
       if not expected:
           return  # dev / unset token: open gate
       if creds is None or creds.credentials != expected:
           raise HTTPException(
               status_code=status.HTTP_401_UNAUTHORIZED,
               detail="invalid or missing bearer token",
               headers={"WWW-Authenticate": "Bearer"},
           )
   ```
