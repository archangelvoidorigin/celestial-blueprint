## Context
Because markdown is canonical and SQLite is a rebuildable index, the system must export a blueprint to markdown (from the DB) and import markdown back into the DB — and rebuild the entire DB from the `blueprints/` directory after a DB loss (the recovery guarantee, tested in T1.8.02). This router wires the parser (T1.2.05), serializer (T1.2.06), store (T1.3.05), and file sync (T1.4.02) into three endpoints: export, import, rebuild.

## Objective
Create `api/io.py` with export (DB→md), import (md→DB), and rebuild (dir→DB), and register it in `main.py`.

## Instructions
1. Create `backend/app/api/io.py` with exactly:
   ```python
   """Import/export: markdown <-> DB. Markdown is canonical."""
   from __future__ import annotations

   from pathlib import Path

   from fastapi import APIRouter, HTTPException
   from fastapi.responses import PlainTextResponse
   from pydantic import BaseModel

   from app.core.config import settings
   from app.services import store
   from app.services.files import write_blueprint_file
   from app.services.parser import parse_blueprint
   from app.services.serializer import serialize_blueprint

   router = APIRouter(prefix="/api", tags=["io"])


   class ImportIn(BaseModel):
       markdown: str


   @router.get("/blueprints/{bid}/export", response_class=PlainTextResponse)
   def export_blueprint(bid: str):
       bp = store.get_blueprint(bid)
       if bp is None:
           raise HTTPException(status_code=404, detail="blueprint not found")
       return serialize_blueprint(bp)


   @router.post("/blueprints/import")
   def import_blueprint(body: ImportIn):
       try:
           bp = parse_blueprint(body.markdown)
       except ValueError as e:
           raise HTTPException(status_code=400, detail=f"parse error: {e}")
       bid = store.save_blueprint(bp)
       write_blueprint_file(bid)
       return store.get_blueprint(bid)


   @router.post("/rebuild")
   def rebuild_from_disk():
       """Rebuild the DB index from every .md in the blueprints directory."""
       d = Path(settings.blueprints_dir)
       count = 0
       for path in sorted(d.glob("*.md")):
           bp = parse_blueprint(path.read_text())
           store.save_blueprint(bp)
           count += 1
       return {"imported": count}
   ```
2. Edit `backend/app/main.py`: under `# --- ROUTER INCLUDES ---`, ADD:
   ```python
   from app.api.io import router as io_router
   app.include_router(io_router)
   ```
