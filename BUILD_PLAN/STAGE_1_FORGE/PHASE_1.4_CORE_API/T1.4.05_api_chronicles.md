# TASK T1.4.05 — `app/api/chronicles.py` (add / list chronicle)

- **Stage / Phase**: Stage 1 — Forge / Phase 1.4
- **Depends on**: T1.4.01, T1.3.05
- **Produces**: `backend/app/api/chronicles.py`, edit to `backend/app/main.py`

## Context
Chronicles are the session journal (RAW entries at breakpoints, CYCLE entries at session end — see the methodology in Phase 1.5). Each chronicle is written both as a markdown file under `chronicles/` (raw or cycles/<n>) and as a DB record via `store.add_chronicle` / `store.list_chronicles` (T1.3.05). This router exposes add + list. The chronicles directory comes from `settings.chronicles_dir` (T1.1.04).

## Objective
Create `api/chronicles.py` with add + list endpoints that also persist a markdown file, and register it in `main.py`.

## Instructions
1. Create `backend/app/api/chronicles.py` with exactly:
   ```python
   """Chronicle endpoints: session journal (raw + cycle)."""
   from __future__ import annotations

   from datetime import datetime
   from pathlib import Path

   from fastapi import APIRouter, HTTPException
   from pydantic import BaseModel

   from app.core.config import settings
   from app.services import store

   router = APIRouter(prefix="/api/blueprints", tags=["chronicles"])


   class ChronicleIn(BaseModel):
       kind: str = "raw"          # raw | cycle
       trigger: str = ""
       cycle: str = ""
       body: str = ""


   @router.post("/{bid}/chronicles")
   def add_chronicle(bid: str, item: ChronicleIn):
       if not store.blueprint_exists(bid):
           raise HTTPException(status_code=404, detail="blueprint not found")
       stamp = datetime.now().strftime("%Y-%m-%d-%H%M%S")
       subdir = "raw" if item.kind == "raw" else f"cycles/{item.cycle or '0'}"
       path = Path(settings.chronicles_dir) / subdir / f"{stamp}-{item.kind}.md"
       path.parent.mkdir(parents=True, exist_ok=True)
       path.write_text(item.body)
       ch_id = store.add_chronicle(bid, {
           "kind": item.kind, "trigger": item.trigger,
           "cycle": item.cycle, "path": str(path), "body": item.body,
       })
       return {"id": ch_id, "path": str(path)}


   @router.get("/{bid}/chronicles")
   def list_chronicles(bid: str):
       if not store.blueprint_exists(bid):
           raise HTTPException(status_code=404, detail="blueprint not found")
       return store.list_chronicles(bid)
   ```
2. Edit `backend/app/main.py`: under `# --- ROUTER INCLUDES ---`, ADD:
   ```python
   from app.api.chronicles import router as chronicles_router
   app.include_router(chronicles_router)
   ```
