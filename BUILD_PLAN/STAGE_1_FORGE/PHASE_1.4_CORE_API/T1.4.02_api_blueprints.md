## Context
This task adds the blueprint endpoints (create/list/get/search/patch/archive) and the shared markdown-file-sync helper every mutating endpoint uses to honour the round-trip contract (mutations always rewrite the `.md` file — see `docs/FORMAT.md`). `files.py` writes a blueprint's markdown from the DB using the serializer (T1.2.06); the routers in T1.4.03–05 reuse it. The app is wired by appending one include line to `main.py`.

## Objective
Create `files.py` (markdown sync), `api/blueprints.py` (the blueprint router), and register the router in `main.py`.

## Instructions
1. Create `backend/app/services/files.py` with exactly:
   ```python
   """Markdown file sync: DB blueprint -> canonical .md on disk."""
   from __future__ import annotations

   from pathlib import Path

   from app.core.config import settings
   from app.services.serializer import serialize_blueprint
   from app.services.store import get_blueprint


   def blueprint_path(bid: str) -> Path:
       return Path(settings.blueprints_dir) / f"{bid}.md"


   def write_blueprint_file(bid: str) -> str:
       """Regenerate the markdown file for a blueprint from the DB."""
       bp = get_blueprint(bid)
       if bp is None:
           raise ValueError(f"blueprint not found: {bid}")
       path = blueprint_path(bid)
       path.parent.mkdir(parents=True, exist_ok=True)
       path.write_text(serialize_blueprint(bp))
       return str(path)
   ```
2. Create `backend/app/api/blueprints.py` with exactly:
   ```python
   """Blueprint endpoints."""
   from __future__ import annotations

   from fastapi import APIRouter, HTTPException

   from app.models.schemas import BlueprintCreate, BlueprintPatch
   from app.services import store
   from app.services.files import write_blueprint_file

   router = APIRouter(prefix="/api/blueprints", tags=["blueprints"])


   @router.post("")
   def create_blueprint(data: BlueprintCreate):
       bid = store.create_blueprint(data)
       write_blueprint_file(bid)
       return store.get_blueprint(bid)


   @router.get("")
   def list_blueprints():
       return store.list_blueprints()


   @router.get("/search")
   def search_blueprints(q: str = ""):
       ql = q.lower()
       return [b for b in store.list_blueprints() if ql in b["name"].lower()]


   @router.get("/{bid}")
   def get_blueprint(bid: str):
       bp = store.get_blueprint(bid)
       if bp is None:
           raise HTTPException(status_code=404, detail="blueprint not found")
       return bp


   @router.patch("/{bid}")
   def patch_blueprint(bid: str, body: BlueprintPatch):
       if not store.patch_blueprint(bid, body):
           raise HTTPException(status_code=404, detail="blueprint not found")
       write_blueprint_file(bid)
       return store.get_blueprint(bid)


   @router.post("/{bid}/archive")
   def archive_blueprint(bid: str):
       if not store.archive_blueprint(bid):
           raise HTTPException(status_code=404, detail="blueprint not found")
       write_blueprint_file(bid)
       return {"status": "archived", "id": bid}
   ```
