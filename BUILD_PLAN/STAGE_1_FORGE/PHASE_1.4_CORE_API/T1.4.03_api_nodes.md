# TASK T1.4.03 — `app/api/nodes.py` (nodes + edges)

- **Stage / Phase**: Stage 1 — Forge / Phase 1.4
- **Depends on**: T1.4.01, T1.4.02, T1.3.05
- **Produces**: `backend/app/api/nodes.py`, edit to `backend/app/main.py`

## Context
Nodes are the canvas cards; edges connect them. This router adds endpoints to add a node and add an edge to an existing blueprint, persisting via the store (T1.3.05) and rewriting the markdown via `write_blueprint_file` (T1.4.02). Bodies validate against the `Node` and `Edge` Pydantic models (T1.3.03). Every mutation returns the updated full blueprint so the client can refresh.

## Objective
Create `api/nodes.py` with add-node and add-edge endpoints and register it in `main.py`.

## Instructions
1. Create `backend/app/api/nodes.py` with exactly:
   ```python
   """Node and edge endpoints."""
   from __future__ import annotations

   from fastapi import APIRouter, HTTPException

   from app.models.schemas import Edge, Node
   from app.services import store
   from app.services.files import write_blueprint_file

   router = APIRouter(prefix="/api/blueprints", tags=["nodes"])


   @router.post("/{bid}/nodes")
   def add_node(bid: str, node: Node):
       if not store.blueprint_exists(bid):
           raise HTTPException(status_code=404, detail="blueprint not found")
       store.add_node(bid, node.model_dump())
       write_blueprint_file(bid)
       return store.get_blueprint(bid)


   @router.post("/{bid}/edges")
   def add_edge(bid: str, edge: Edge):
       if not store.blueprint_exists(bid):
           raise HTTPException(status_code=404, detail="blueprint not found")
       store.add_edge(bid, edge.model_dump())
       write_blueprint_file(bid)
       return store.get_blueprint(bid)
   ```
2. Edit `backend/app/main.py`: under the existing `# --- ROUTER INCLUDES ---` block, after the blueprints include, ADD:
   ```python
   from app.api.nodes import router as nodes_router
   app.include_router(nodes_router)
   ```

## Acceptance Criteria
