# TASK T1.3.02 — `app/core/db.py` connection + init (WAL, FK)

- **Stage / Phase**: Stage 1 — Forge / Phase 1.3
- **Depends on**: T1.3.01
- **Produces**: `backend/app/core/db.py`

## Context
The store and API layers need a single, consistent way to open the SQLite database with the right pragmas (WAL for write concurrency, foreign keys ON) and to initialise the schema on first run. The DB path comes from `settings.db_path` (T1.1.04). The schema is `app/models/schema.sql` (T1.3.01). Rows should come back as dict-like objects (`sqlite3.Row`).

## Objective
Implement `get_conn()`, `init_db()`, and a `connection()` context manager in `db.py`.

## Instructions
1. Create `backend/app/core/db.py` with exactly:
   ```python
   """SQLite connection + initialisation. WAL mode, foreign keys on."""
   from __future__ import annotations

   import sqlite3
   from contextlib import contextmanager
   from pathlib import Path

   from app.core.config import settings

   SCHEMA_PATH = Path(__file__).resolve().parents[1] / "models" / "schema.sql"


   def get_conn() -> sqlite3.Connection:
       """Open a configured connection. Caller is responsible for closing."""
       db_path = Path(settings.db_path)
       db_path.parent.mkdir(parents=True, exist_ok=True)
       conn = sqlite3.connect(str(db_path))
       conn.row_factory = sqlite3.Row
       conn.execute("PRAGMA journal_mode = WAL;")
       conn.execute("PRAGMA foreign_keys = ON;")
       return conn


   def init_db() -> None:
       """Create tables if they do not exist (idempotent)."""
       schema = SCHEMA_PATH.read_text()
       conn = get_conn()
       try:
           conn.executescript(schema)
           conn.commit()
       finally:
           conn.close()


   @contextmanager
   def connection():
       """Context manager yielding a connection, committing on success."""
       conn = get_conn()
       try:
           yield conn
           conn.commit()
       except Exception:
           conn.rollback()
           raise
       finally:
           conn.close()
   ```
