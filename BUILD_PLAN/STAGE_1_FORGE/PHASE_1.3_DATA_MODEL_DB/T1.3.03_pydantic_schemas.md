## Context
The API and store layers exchange typed objects, not loose dicts. These Pydantic v2 models mirror the canonical blueprint dict (parser shape, T1.2.03) and the SQLite tables (T1.3.01). They define the request/response contract the API (Phase 1.4) and the TypeScript types (T1.6.02) follow. Pydantic 2.x is already installed (T1.1.02).

## Objective
Define Pydantic models for every blueprint entity plus create/update request models.

## Instructions
1. Create `backend/app/models/schemas.py` with exactly:
   ```python
   """Pydantic models mirroring the canonical blueprint dict and SQLite schema."""
   from __future__ import annotations

   from typing import Any, Optional

   from pydantic import BaseModel, Field


   class Category(BaseModel):
       id: str
       name: str


   class Node(BaseModel):
       id: str
       cat: str = ""
       type: str
       title: str = ""
       body: str = ""
       data: dict[str, Any] = Field(default_factory=dict)


   class Edge(BaseModel):
       src: str
       dst: str
       label: str = ""


   class Assumption(BaseModel):
       id: str
       bp_ref: str = ""
       status: str = "unvalidated"
       risk: str = "med"
       body: str = ""


   class Decision(BaseModel):
       id: str
       bp_ref: str = ""
       status: str = "deferred"
       date: str = ""
       body: str = ""


   class Risk(BaseModel):
       id: str
       risk: str = ""
       likelihood: str = ""
       impact: str = ""
       mitigation: str = ""
       status: str = "open"


   class Research(BaseModel):
       id: str
       query: str = ""
       date: str = ""
       body: str = ""


   class ChronicleLink(BaseModel):
       id: str
       type: str = "raw"
       path: str = ""


   class Version(BaseModel):
       version: str
       date: str = ""
       note: str = ""


   class Blueprint(BaseModel):
       """Full blueprint = the canonical dict, typed."""
       frontmatter: dict[str, Any] = Field(default_factory=dict)
       summary: str = ""
       categories: list[Category] = Field(default_factory=list)
       nodes: list[Node] = Field(default_factory=list)
       assumptions: list[Assumption] = Field(default_factory=list)
       decisions: list[Decision] = Field(default_factory=list)
       risks: list[Risk] = Field(default_factory=list)
       research: list[Research] = Field(default_factory=list)
       chronicle_links: list[ChronicleLink] = Field(default_factory=list)
       edges: list[Edge] = Field(default_factory=list)
       versions: list[Version] = Field(default_factory=list)


   class BlueprintSummary(BaseModel):
       """Lightweight row for list views."""
       id: str
       name: str
       version: str
       tier: str
       status: str
       updated_at: str = ""


   class BlueprintCreate(BaseModel):
       name: str
       tier: str = "small"
       realm: str = ""
       summary: str = ""


   class BlueprintPatch(BaseModel):
       name: Optional[str] = None
       status: Optional[str] = None
       version: Optional[str] = None
       summary: Optional[str] = None
   ```
