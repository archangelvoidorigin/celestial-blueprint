# TASK T1.3.01 — `app/models/schema.sql` (8 tables)

- **Stage / Phase**: Stage 1 — Forge / Phase 1.3
- **Depends on**: T1.1.04
- **Produces**: `backend/app/models/schema.sql`

## Context
SQLite is the rebuildable query index over the canonical blueprint markdown (see `docs/FORMAT.md`). This task writes the schema: 8 tables (blueprints, nodes, edges, assumptions, decisions, risks, chronicles, versions). Sections that are reference-only for round-trip (categories, research log, chronicle links) are stored as JSON columns on `blueprints` rather than as their own tables. Foreign keys cascade on blueprint delete; child rows are scoped by `blueprint_id`.

## Objective
Write `schema.sql` creating the 8 tables with foreign keys and helpful indexes.

## Instructions
1. Create `backend/app/models/schema.sql` with exactly:
   ```sql
   -- Celestial Blueprint schema. SQLite is a rebuildable index over markdown.
   PRAGMA foreign_keys = ON;

   CREATE TABLE IF NOT EXISTS blueprints (
       id                   TEXT PRIMARY KEY,
       name                 TEXT NOT NULL,
       version              TEXT NOT NULL DEFAULT '0.1.0',
       tier                 TEXT NOT NULL DEFAULT 'small',
       status               TEXT NOT NULL DEFAULT 'draft',
       realm                TEXT,
       summary              TEXT,
       frontmatter_json     TEXT NOT NULL DEFAULT '{}',
       categories_json      TEXT NOT NULL DEFAULT '[]',
       research_json        TEXT NOT NULL DEFAULT '[]',
       chronicle_links_json TEXT NOT NULL DEFAULT '[]',
       created_at           TEXT NOT NULL DEFAULT (datetime('now')),
       updated_at           TEXT NOT NULL DEFAULT (datetime('now'))
   );

   CREATE TABLE IF NOT EXISTS nodes (
       blueprint_id  TEXT NOT NULL,
       id            TEXT NOT NULL,
       cat           TEXT,
       type          TEXT NOT NULL,
       title         TEXT,
       body          TEXT,
       data_json     TEXT NOT NULL DEFAULT '{}',
       PRIMARY KEY (blueprint_id, id),
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS edges (
       blueprint_id  TEXT NOT NULL,
       src           TEXT NOT NULL,
       dst           TEXT NOT NULL,
       label         TEXT,
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS assumptions (
       blueprint_id  TEXT NOT NULL,
       id            TEXT NOT NULL,
       bp_ref        TEXT,
       status        TEXT,
       risk          TEXT,
       body          TEXT,
       PRIMARY KEY (blueprint_id, id),
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS decisions (
       blueprint_id  TEXT NOT NULL,
       id            TEXT NOT NULL,
       bp_ref        TEXT,
       status        TEXT,
       date          TEXT,
       body          TEXT,
       PRIMARY KEY (blueprint_id, id),
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS risks (
       blueprint_id  TEXT NOT NULL,
       id            TEXT NOT NULL,
       risk          TEXT,
       likelihood    TEXT,
       impact        TEXT,
       mitigation    TEXT,
       status        TEXT,
       PRIMARY KEY (blueprint_id, id),
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS chronicles (
       id            INTEGER PRIMARY KEY AUTOINCREMENT,
       blueprint_id  TEXT NOT NULL,
       kind          TEXT NOT NULL DEFAULT 'raw',   -- raw | cycle
       trigger       TEXT,
       cycle         TEXT,
       path          TEXT,
       body          TEXT,
       created_at    TEXT NOT NULL DEFAULT (datetime('now')),
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE TABLE IF NOT EXISTS versions (
       blueprint_id  TEXT NOT NULL,
       version       TEXT NOT NULL,
       date          TEXT,
       note          TEXT,
       FOREIGN KEY (blueprint_id) REFERENCES blueprints(id) ON DELETE CASCADE
   );

   CREATE INDEX IF NOT EXISTS idx_nodes_bp   ON nodes(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_edges_bp   ON edges(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_assum_bp   ON assumptions(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_dec_bp     ON decisions(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_risks_bp   ON risks(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_chron_bp   ON chronicles(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_versions_bp ON versions(blueprint_id);
   CREATE INDEX IF NOT EXISTS idx_blueprints_tier ON blueprints(tier);
   CREATE INDEX IF NOT EXISTS idx_blueprints_status ON blueprints(status);
   ```
