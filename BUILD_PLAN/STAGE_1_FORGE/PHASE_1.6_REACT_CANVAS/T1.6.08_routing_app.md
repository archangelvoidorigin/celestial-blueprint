# TASK T1.6.08 — Routing + `App.tsx` layout wiring

- **Stage / Phase**: Stage 1 — Forge / Phase 1.6
- **Depends on**: T1.6.03, T1.6.04
- **Produces**: `frontend/src/App.tsx` (rewrite), `frontend/src/main.tsx` (minor edit to wrap with Router)

## Context
All pages and the router are now built. This task wires them together: `App.tsx` becomes the route tree (`/` → `BlueprintList`, `/blueprint/:id` → `Canvas`), and `main.tsx` wraps the app in `<BrowserRouter>`. React Router DOM is already installed (T1.1.05). After this task, running `npm run dev` serves the full app.

## Objective
Rewrite `App.tsx` with the route tree and update `main.tsx` to add `BrowserRouter`.

## Instructions
1. Rewrite `frontend/src/App.tsx` with exactly:
   ```tsx
   import { Routes, Route, Navigate } from "react-router-dom";
   import BlueprintList from "./pages/BlueprintList";
   import Canvas from "./pages/Canvas";

   export default function App() {
     return (
       <Routes>
         <Route path="/" element={<BlueprintList />} />
         <Route path="/blueprint/:id" element={<Canvas />} />
         <Route path="*" element={<Navigate to="/" replace />} />
       </Routes>
     );
   }
   ```
2. Rewrite `frontend/src/main.tsx` with exactly:
   ```tsx
   import React from "react";
   import ReactDOM from "react-dom/client";
   import { BrowserRouter } from "react-router-dom";
   import App from "./App";
   import "./index.css";

   ReactDOM.createRoot(document.getElementById("root")!).render(
     <React.StrictMode>
       <BrowserRouter>
         <App />
       </BrowserRouter>
     </React.StrictMode>
   );
   ```

## Acceptance Criteria
- [ ] `App.tsx` has exactly three routes: `/`, `/blueprint/:id`, and `*` redirect
- [ ] `main.tsx` wraps the app in `<BrowserRouter>`
- [ ] `npm run build` succeeds with all pages reachable
- [ ] Unknown paths redirect to `/`
