# TASK T1.5.07 — `link_skill.sh` symlink installer

- **Stage / Phase**: Stage 1 — Forge / Phase 1.5
- **Depends on**: T1.5.06
- **Produces**: `skill/celestial-blueprint/link_skill.sh`

## Context
The skill lives in `skill/celestial-blueprint/` inside the project. To load it in Claude Code, OpenCode, and other agents, it needs to be symlinked into each tool's skills directory. This script creates those symlinks idempotently. The user runs it once after cloning; it does not copy — changes to `SKILL.md` in the project are immediately reflected everywhere. Per the project rule: all real files stay inside CHAOS_DAO_FORGE; external dirs hold only symlinks.

## Objective
Create `link_skill.sh` that symlinks the skill into Claude Code, OpenCode, and OpenAI Codex skill directories.

## Instructions
1. Create `skill/celestial-blueprint/link_skill.sh` with exactly:
   ```bash
   #!/usr/bin/env bash
   # Symlink celestial-blueprint skill into each tool's skills directory.
   # Run from any directory; idempotent.
   set -euo pipefail

   SKILL_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
   SKILL_NAME="celestial-blueprint"

   link_skill() {
       local target_dir="$1"
       if [ ! -d "$(dirname "$target_dir")" ]; then
           echo "  SKIP $target_dir (parent dir absent)"
           return
       fi
       mkdir -p "$target_dir"
       local dest="$target_dir/$SKILL_NAME"
       if [ -L "$dest" ]; then
           echo "  EXISTS $dest (already linked)"
       elif [ -e "$dest" ]; then
           echo "  SKIP $dest (non-symlink exists — remove manually)"
       else
           ln -s "$SKILL_DIR" "$dest"
           echo "  LINKED $dest"
       fi
   }

   echo "Linking celestial-blueprint skill..."
   link_skill "$HOME/.claude/skills"
   link_skill "$HOME/.config/opencode/skills"
   link_skill "$HOME/.agents/skills"
   link_skill "$HOME/.opencode/skills"
   echo "Done. Restart your agent tool to load the skill."
   ```
2. Make it executable:
   ```bash
   chmod +x /home/void-yggdrasil/CHAOS_DAO_FORGE/00_CORTEX/01_CREATION_REALM/Celestial_Blueprint/skill/celestial-blueprint/link_skill.sh
   ```
