# TASK T1.1.05 — Frontend Vite + React + TypeScript init

- **Stage / Phase**: Stage 1 — Forge / Phase 1.1
- **Depends on**: T1.1.01
- **Produces**: `frontend/package.json`, `frontend/vite.config.ts`, `frontend/tsconfig.json`, `frontend/tsconfig.node.json`, `frontend/.eslintrc.cjs` (optional skip ok)

## Context
The frontend is a Vite + React 18 + TypeScript single-page app. The `frontend/` directory and its `src/` subtree already exist (from T1.1.01). This task adds the project manifest and TypeScript/Vite config so `npm install` and `npm run dev` work. The dev server runs on port 5173 and proxies `/api` to the backend on port 8787 so the browser can call the API without CORS friction in dev.

## Objective
Create the frontend `package.json` and Vite/TypeScript config files (no source components yet — those come in T1.1.07).

## Instructions
1. Create `frontend/package.json` with exactly:
   ```json
   {
     "name": "celestial-frontend",
     "private": true,
     "version": "0.1.0",
     "type": "module",
     "scripts": {
       "dev": "vite",
       "build": "tsc && vite build",
       "preview": "vite preview"
     },
     "dependencies": {
       "react": "^18.3.1",
       "react-dom": "^18.3.1",
       "react-router-dom": "^6.26.0",
       "reactflow": "^11.11.4"
     },
     "devDependencies": {
       "@types/react": "^18.3.3",
       "@types/react-dom": "^18.3.0",
       "@vitejs/plugin-react": "^4.3.1",
       "typescript": "^5.5.4",
       "vite": "^5.4.0"
     }
   }
   ```
2. Create `frontend/vite.config.ts` with exactly:
   ```ts
   import { defineConfig } from "vite";
   import react from "@vitejs/plugin-react";

   export default defineConfig({
     plugins: [react()],
     server: {
       port: 5173,
       proxy: {
         "/api": {
           target: "http://localhost:8787",
           changeOrigin: true,
         },
       },
     },
   });
   ```
3. Create `frontend/tsconfig.json` with exactly:
   ```json
   {
     "compilerOptions": {
       "target": "ES2020",
       "useDefineForClassFields": true,
       "lib": ["ES2020", "DOM", "DOM.Iterable"],
       "module": "ESNext",
       "skipLibCheck": true,
       "moduleResolution": "bundler",
       "allowImportingTsExtensions": true,
       "resolveJsonModule": true,
       "isolatedModules": true,
       "noEmit": true,
       "jsx": "react-jsx",
       "strict": true,
       "noUnusedLocals": true,
       "noUnusedParameters": true,
       "noFallthroughCasesInSwitch": true,
       "baseUrl": ".",
       "paths": { "@/*": ["src/*"] }
     },
     "include": ["src"],
     "references": [{ "path": "./tsconfig.node.json" }]
   }
   ```
4. Create `frontend/tsconfig.node.json` with exactly:
   ```json
   {
     "compilerOptions": {
       "composite": true,
       "skipLibCheck": true,
       "module": "ESNext",
       "moduleResolution": "bundler",
       "allowSyntheticDefaultImports": true,
       "strict": true
     },
     "include": ["vite.config.ts"]
   }
   ```
