# TASK T1.1.08 — Root `Makefile` + `dev.fish` run script

- **Stage / Phase**: Stage 1 — Forge / Phase 1.1
- **Depends on**: T1.1.04, T1.1.07
- **Produces**: `Makefile` (project root), `dev.fish` (project root)

## Context
The backend (FastAPI on port 8787) and frontend (Vite on port 5173) are now both runnable. Developers need one-command entry points to install deps and run each service. This task adds a `Makefile` with the common targets and a `dev.fish` script (the user's shell is fish) that launches backend and frontend together. The project root is `Celestial_Blueprint/`.

## Objective
Create a root `Makefile` and a `dev.fish` script that install dependencies and run the two services.

## Instructions
1. Create `Makefile` at the **project root** (`Celestial_Blueprint/Makefile`) with exactly (NOTE: recipe lines MUST be indented with a TAB, not spaces):
   ```make
   ROOT := $(shell pwd)
   BACKEND := $(ROOT)/backend
   FRONTEND := $(ROOT)/frontend

   .PHONY: install install-backend install-frontend backend frontend dev test clean

   install: install-backend install-frontend

   install-backend:
   	cd $(BACKEND) && python -m venv .venv && . .venv/bin/activate && pip install -q -r requirements.txt

   install-frontend:
   	cd $(FRONTEND) && npm install

   backend:
   	cd $(BACKEND) && . .venv/bin/activate && uvicorn app.main:app --reload --port 8787

   frontend:
   	cd $(FRONTEND) && npm run dev

   dev:
   	fish $(ROOT)/dev.fish

   test:
   	cd $(BACKEND) && . .venv/bin/activate && pytest -q

   clean:
   	rm -rf $(BACKEND)/.venv $(FRONTEND)/node_modules $(FRONTEND)/dist
   ```
2. Create `dev.fish` at the **project root** (`Celestial_Blueprint/dev.fish`) with exactly:
   ```fish
   #!/usr/bin/env fish
   # Run Celestial Blueprint backend + frontend together (dev).
   set ROOT (cd (dirname (status -f)); and pwd)
   set BACKEND $ROOT/backend
   set FRONTEND $ROOT/frontend

   function _cleanup --on-signal INT --on-signal TERM
       echo "Stopping..."
       kill $BACKEND_PID $FRONTEND_PID 2>/dev/null
       exit 0
   end

   echo "Starting backend on :8787 ..."
   cd $BACKEND
   source .venv/bin/activate.fish 2>/dev/null
   uvicorn app.main:app --reload --port 8787 &
   set BACKEND_PID $last_pid

   echo "Starting frontend on :5173 ..."
   cd $FRONTEND
   npm run dev &
   set FRONTEND_PID $last_pid

   echo "Backend PID $BACKEND_PID | Frontend PID $FRONTEND_PID"
   echo "Open http://localhost:5173 — Ctrl-C to stop both."
   wait
   ```
3. Make `dev.fish` executable: `chmod +x dev.fish`.
