# TASK T1.1.07 — Frontend base shell (`index.html`, `main.tsx`, `App.tsx`)

- **Stage / Phase**: Stage 1 — Forge / Phase 1.1
- **Depends on**: T1.1.06
- **Produces**: `frontend/index.html`, `frontend/src/main.tsx`, `frontend/src/App.tsx`, `frontend/src/vite-env.d.ts`

## Context
The frontend has its build pipeline (Vite/TS from T1.1.05) and styling (Tailwind/React Flow from T1.1.06) but no entry point yet. This task adds the minimal React shell: the HTML host page, the React root that mounts into it, and a placeholder `App` component proving the dark theme renders. Real pages and the canvas come in Phase 1.6; this is just a runnable skeleton.

## Objective
Create the HTML entry, React mount point, and a placeholder App so `npm run dev` serves a styled page.

## Instructions
1. Create `frontend/index.html` with exactly:
   ```html
   <!doctype html>
   <html lang="en">
     <head>
       <meta charset="UTF-8" />
       <meta name="viewport" content="width=device-width, initial-scale=1.0" />
       <title>Celestial Blueprint</title>
     </head>
     <body>
       <div id="root"></div>
       <script type="module" src="/src/main.tsx"></script>
     </body>
   </html>
   ```
2. Create `frontend/src/vite-env.d.ts` with exactly:
   ```ts
   /// <reference types="vite/client" />
   ```
3. Create `frontend/src/main.tsx` with exactly:
   ```tsx
   import React from "react";
   import ReactDOM from "react-dom/client";
   import App from "./App";
   import "./index.css";

   ReactDOM.createRoot(document.getElementById("root")!).render(
     <React.StrictMode>
       <App />
     </React.StrictMode>
   );
   ```
4. Create `frontend/src/App.tsx` with exactly:
   ```tsx
   export default function App() {
     return (
       <div className="flex h-full items-center justify-center bg-bg text-ink">
         <div className="rounded-lg border border-border bg-surface px-8 py-6 text-center">
           <h1 className="text-2xl font-semibold text-accent">Celestial Blueprint</h1>
           <p className="mt-2 text-sm text-muted">Frontend shell online.</p>
         </div>
       </div>
     );
   }
   ```
