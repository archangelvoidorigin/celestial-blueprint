# TASK T1.1.02 — Create backend dependency files

- **Stage / Phase**: Stage 1 — Forge / Phase 1.1
- **Depends on**: T1.1.01
- **Produces**: `backend/requirements.txt`, `backend/pyproject.toml`

## Context
The backend is a FastAPI service. The `backend/` directory exists (from T1.1.01). This task declares its Python dependencies so the venv can be built. Note `sqlite3` is part of the Python standard library and is NOT listed here.

## Objective
Create `requirements.txt` and `pyproject.toml` pinning the backend dependencies.

## Instructions
1. Create `backend/requirements.txt` with exactly:
   ```
   fastapi==0.115.*
   uvicorn[standard]==0.32.*
   pydantic==2.*
   pyyaml==6.*
   pytest==8.*
   httpx==0.27.*
   ```
2. Create `backend/pyproject.toml` with exactly:
   ```toml
   [project]
   name = "celestial-backend"
   version = "0.1.0"
   description = "Celestial Blueprint planning backend (API-first)"
   requires-python = ">=3.11"
   dependencies = [
       "fastapi>=0.115,<0.116",
       "uvicorn[standard]>=0.32,<0.33",
       "pydantic>=2,<3",
       "pyyaml>=6,<7",
   ]

   [project.optional-dependencies]
   dev = ["pytest>=8,<9", "httpx>=0.27,<0.28"]

   [tool.pytest.ini_options]
   testpaths = ["tests"]
   pythonpath = ["."]
   ```

## Acceptance Criteria
- [ ] Both files exist in `backend/`
- [ ] A fresh venv installs all deps and imports succeed
