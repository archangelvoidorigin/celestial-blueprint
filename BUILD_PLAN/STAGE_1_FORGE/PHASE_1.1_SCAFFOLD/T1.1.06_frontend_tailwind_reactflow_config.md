# TASK T1.1.06 — Frontend Tailwind + React Flow config

- **Stage / Phase**: Stage 1 — Forge / Phase 1.1
- **Depends on**: T1.1.05
- **Produces**: `frontend/tailwind.config.js`, `frontend/postcss.config.js`, `frontend/src/index.css`, updated `frontend/package.json` (adds tailwind devDeps)

## Context
The UI uses Tailwind CSS for styling and React Flow for the canvas. T1.1.05 created the Vite/React/TS project. This task installs and configures Tailwind (via PostCSS) and writes the global stylesheet that pulls in Tailwind layers plus React Flow's base CSS. The theme is dark with a dot-grid canvas (the canvas styling itself comes later in Phase 1.6; here we only set up the pipeline and dark base tokens).

## Objective
Add Tailwind + PostCSS config and the global `index.css` so utility classes and React Flow styles are available app-wide.

## Instructions
1. Add Tailwind dev dependencies. Edit `frontend/package.json` `devDependencies` to ADD these three entries (keep the existing ones):
   ```json
   "autoprefixer": "^10.4.20",
   "postcss": "^8.4.41",
   "tailwindcss": "^3.4.10"
   ```
2. Create `frontend/tailwind.config.js` with exactly:
   ```js
   /** @type {import('tailwindcss').Config} */
   export default {
     content: ["./index.html", "./src/**/*.{ts,tsx}"],
     theme: {
       extend: {
         colors: {
           bg: "#0b0e14",
           surface: "#11151f",
           border: "#1e2430",
           ink: "#e6e9ef",
           muted: "#8b93a7",
           accent: "#7c5cff",
         },
       },
     },
     plugins: [],
   };
   ```
3. Create `frontend/postcss.config.js` with exactly:
   ```js
   export default {
     plugins: {
       tailwindcss: {},
       autoprefixer: {},
     },
   };
   ```
4. Create `frontend/src/index.css` with exactly:
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;

   /* React Flow base styles */
   @import "reactflow/dist/style.css";

   :root {
     color-scheme: dark;
   }

   html,
   body,
   #root {
     height: 100%;
     margin: 0;
   }

   body {
     background-color: #0b0e14;
     color: #e6e9ef;
     font-family: ui-sans-serif, system-ui, -apple-system, "Segoe UI", sans-serif;
   }
   ```
5. Run `npm install` in `frontend/` to fetch the new Tailwind deps.

## Acceptance Criteria
- [ ] `tailwind.config.js`, `postcss.config.js`, `src/index.css` all exist
- [ ] `package.json` lists `tailwindcss`, `postcss`, `autoprefixer` in devDependencies
- [ ] `npm install` completes; tailwindcss resolves
- [ ] `index.css` imports Tailwind layers AND `reactflow/dist/style.css`
